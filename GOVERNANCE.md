# Governance

The governance model for RCP can be found online at
[the Governance page](https://rcp-6d6ef6d5.mintlify.site/community/governance).

In short: RCP is stewarded by the **RCP Working Group** and evolves through
public RFDs (Requests for Discussion) on GitHub. Anyone may open an RFD; changes
to the normative spec or schema require Working Group consensus and land only in
a numbered protocol version. See the linked page for the full process, roles,
and decision rules.
