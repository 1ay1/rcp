# Maintainers

RCP is stewarded by the **RCP Working Group**. Maintainers review RFDs, cut
protocol versions, and keep the spec, schema, SDKs, and docs in sync.

| Area | Maintainer |
|------|------------|
| Specification & schema | RCP Working Group |
| SDKs (C++/Python/Node/Rust) | RCP Working Group |
| Documentation site | RCP Working Group |

To become a maintainer, establish a track record of high-quality RFDs, reviews,
and SDK/conformance contributions, then ask an existing maintainer to nominate
you. See [Governance](https://rcp-6d6ef6d5.mintlify.site/community/governance)
for the full process.

Security reports should **not** be filed as public issues — see
[SECURITY.md](SECURITY.md).
